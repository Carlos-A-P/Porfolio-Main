Learn Tailwind CSS

A hands-on introduction to utility-first CSS

üé® What is Tailwind CSS?

Tailwind is a utility-first CSS framework. Instead of writing custom CSS, you apply pre-made classes directly to your HTML elements.

Traditional CSS:

style="color: blue; font-size: 24px; font-weight: bold;"

Tailwind CSS:

class="text-blue-600 text-2xl font-bold"

Text Sizes:

text-xs - Extra small text

text-sm - Small text

text-base - Normal text

text-lg - Large text

text-xl - Extra large text

text-2xl - 2X large text

Text Colors:

text-red-600 - Red text

text-blue-600 - Blue text

text-green-600 - Green text

text-purple-600 - Purple text

Font Weights:

font-light - Light text

font-normal - Normal text

font-bold - Bold text

üìè Spacing (Padding & Margin)

Spacing uses numbers: 1 = 0.25rem (4px), 2 = 0.5rem (8px), 4 = 1rem (16px), etc.

Padding (p):

p-2 (padding all sides)
px-4 py-2 (horizontal & vertical)

Margin (m):

mt-4 (margin top)
mb-4 (margin bottom)

Quick Reference:

  • ‚Ä¢ p-4 = padding all sides
  • ‚Ä¢ pt-4 = padding top
  • ‚Ä¢ px-4 = padding left & right
  • ‚Ä¢ py-4 = padding top & bottom
  • ‚Ä¢ Same works for margin: m-4, mt-4, mx-4, my-4
